## Creating rangamandap website

My friends arebuilding a theater. It will be called Rangamandap and it will be in Nagpur. They have a website skeleton. Some parts are in English and some in hindi. We want it to be available in English, Hindi, and Marathi. The current content can be found at rangamandap.org. For ticketing users should be directed to rangamandap.com.

Start with what is at rangamandap.org and create pages for a trilingual modern website. Take decisions that you need to. It should be elegant, but should not be heavy. You can modify look and feel, but do use the current pictures provided at the website.

We can create the website here under a subdir, and test locally, and perhpas through github pages so that they can see it too and comment.
